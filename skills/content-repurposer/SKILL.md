# Content Repurposer

Transform one long-form piece into multiple platform-specific content derivatives while maintaining voice consistency.

## Atomization Ratio

**Target**: 1 pillar piece → 8+ derivative pieces

From one blog post, generate:
- 3-5 LinkedIn posts (different angles)
- 1 Twitter/X thread (5-10 tweets)
- 2-3 email snippets (newsletter teasers)
- 2-3 ad hooks (paid social)
- 1 short video script (15-60 seconds)

## Platform Specs

### LinkedIn
- 1,300-1,600 characters optimal
- First 140 characters are the hook (before "See more")
- Short paragraphs, line breaks
- 3-5 hashtags at end
- Soft CTA (comment/share) or link in comments

### Twitter/X Thread
- 70-100 characters per tweet optimal
- Number tweets (1/7 format)
- First tweet must stop scroll
- 1-2 hashtags max per thread
- End with CTA + repost request

### Short Video Script (Reels/TikTok/Shorts)
- 15-30 seconds (50-150 words)
- Hook in first 3 seconds
- Structure: Hook → Value → CTA
- Conversational, direct address

### Email/Newsletter Snippet
- 2-3 sentence teaser
- 3-5 key takeaways as bullets
- 1 quotable moment
- Link to full piece CTA

## Hook Formulas

| Type | Formula |
|------|---------|
| Contrarian | "Stop doing [common practice]" |
| Question | "Ever wonder why [problem]?" |
| Number | "5 things about [topic] most people get wrong" |
| Challenge | "Most people get [topic] wrong" |
| Result | "This one change got [result]" |

## Process

1. Analyze source content — key points, structure, main takeaways
2. Identify best hooks for each platform
3. Define voice profile (tone, vocabulary, formality level)
4. Generate all derivatives maintaining voice consistency
5. Deliver complete package ready to publish

## Guidelines

- Provide ready-to-publish copy — no placeholders
- Never change factual claims when repurposing
- Extract direct quotes from source where possible
- If voice guidelines provided, follow strictly
- Ask for source content if not provided
